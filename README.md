# University Department Website

A modern, professional, and fully responsive university department website built with HTML, CSS, and JavaScript. This website showcases a clean academic identity with a consistent color scheme inspired by university branding.

## Features

### 🎨 Design & UI/UX
- **Modern Design**: Clean, minimal, and intuitive layout following modern UI/UX principles
- **University-Inspired Color Scheme**: Professional navy blue, gold accent, and academic blue colors
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Smooth Animations**: Subtle animations and transitions for enhanced user experience
- **Typography**: Professional fonts (Inter and Merriweather) for readability and academic appearance

### 📱 Sections & Pages
- **Hero Section**: Strong visual impact with call-to-action buttons and department statistics
- **About Section**: Department overview, vision, mission, values, and historical timeline
- **Academic Programs**: Interactive tabs for undergraduate, postgraduate, and doctoral programs
- **Faculty Profiles**: Staff directory with qualifications, research interests, and contact information
- **Research Section**: Key research areas, recent publications, and ongoing projects
- **News & Events**: Latest announcements and upcoming events calendar
- **Student Resources**: Downloadable resources, forms, and support services
- **Student Portal**: Login interface mockup with feature highlights
- **Contact Section**: Contact information, contact form, and embedded map

### 🚀 Technical Features
- **Semantic HTML5**: Proper HTML structure for accessibility and SEO
- **CSS3 Features**: Modern CSS with variables, flexbox, grid, and animations
- **JavaScript Interactivity**: Smooth scrolling, form validation, tab navigation, and dynamic content
- **Accessibility**: WCAG compliance with proper ARIA labels, keyboard navigation, and screen reader support
- **SEO Optimized**: Meta tags, semantic structure, and search-friendly content organization
- **Performance**: Lazy loading, optimized animations, and efficient code structure

### 🛠️ Development Highlights
- **Component-Based**: Modular CSS with custom properties for easy maintenance
- **Cross-Browser Compatible**: Works on all modern browsers
- **Progressive Enhancement**: Core functionality works without JavaScript
- **Mobile-First**: Responsive design approach prioritizing mobile experience

- **Primary Navy**: `#1e3a5f` - Professional academic blue

## Getting Started

1. **Download the files** to your local directory
2. **Open `index.html`** in your web browser
3. **Navigate through sections** using the navigation menu or smooth scrolling

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
  --primary-color: #1e3a5f;
  --secondary-color: #3b82f6;
  --accent-color: #f59e0b;
  /* ... other variables */
}
```

### Content
Update the HTML content in `index.html`:
- Replace placeholder text with actual department information
- Update faculty profiles with real staff data
- Modify program details and admission requirements
- Add real news and events

### Images
Replace placeholder images:
- Faculty photos (currently using Picsum Photos)
- Hero background image
- Research project images

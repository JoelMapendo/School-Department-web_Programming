Course Catalog

Explore our comprehensive computer science curriculum

Program Overview

Our Computer Science program offers a comprehensive curriculum designed to prepare students for careers in technology, research, and innovation. Courses are structured to provide both theoretical foundations and practical skills.

100-Level Courses (Foundation)

CS 101: Introduction to Computer Science

CS 101

3 Credits

Fundamental concepts of computer science including algorithms, data structures, and problem-solving methodologies.

Topics Covered:

  • Algorithm design and analysis
  • Basic data structures
  • Programming paradigms
  • Computational thinking
Prerequisite: None Offered: Fall, Spring

CS 102: Programming Fundamentals

CS 102

4 Credits

Introduction to programming using Python. Emphasis on problem-solving and software development practices.

Topics Covered:

  • Variables and data types
  • Control structures
  • Functions and modules
  • Object-oriented programming basics
Prerequisite: CS 101 Offered: Fall, Spring

CS 110: Discrete Mathematics

CS 110

3 Credits

Mathematical foundations essential for computer science including logic, sets, and combinatorics.

Topics Covered:

  • Logic and proofs
  • Sets and functions
  • Combinatorics
  • Graph theory basics
Prerequisite: None Offered: Fall

200-Level Courses (Core)

CS 201: Data Structures and Algorithms

CS 201

4 Credits

In-depth study of data structures and algorithm design and analysis techniques.

Topics Covered:

  • Arrays, linked lists, stacks, queues
  • Trees and graphs
  • Sorting and searching algorithms
  • Algorithm complexity analysis
Prerequisite: CS 102 Offered: Fall, Spring

CS 210: Computer Architecture

CS 210

3 Credits

Study of computer organization, digital logic, and processor design.

Topics Covered:

  • Digital logic circuits
  • Processor architecture
  • Memory systems
  • Instruction sets
Prerequisite: CS 101 Offered: Spring

CS 220: Database Systems

CS 220

3 Credits

Introduction to database design, implementation, and management.

Topics Covered:

  • Relational database design
  • SQL programming
  • Database normalization
  • NoSQL databases
Prerequisite: CS 201 Offered: Fall, Spring

300-Level Courses (Advanced)

CS 301: Software Engineering

CS 301

3 Credits

Principles and practices of software development, testing, and maintenance.

Topics Covered:

  • Software development lifecycle
  • Design patterns
  • Testing methodologies
  • Version control and collaboration
Prerequisite: CS 201 Offered: Fall

CS 310: Artificial Intelligence

CS 310

3 Credits

Introduction to artificial intelligence, machine learning, and intelligent systems.

Topics Covered:

  • Search algorithms
  • Machine learning basics
  • Neural networks
  • Natural language processing
Prerequisite: CS 201 Offered: Spring

CS 320: Computer Networks

CS 320

3 Credits

Study of network protocols, architectures, and distributed systems.

Topics Covered:

  • Network protocols (TCP/IP)
  • Network architectures
  • Network security
  • Distributed systems
Prerequisite: CS 210 Offered: Fall

400-Level Courses (Specialization)

CS 401: Machine Learning

CS 401

3 Credits

Advanced study of machine learning algorithms and applications.

Topics Covered:

  • Supervised and unsupervised learning
  • Deep learning architectures
  • Model evaluation and validation
  • Real-world applications
Prerequisite: CS 310 Offered: Fall

CS 410: Cybersecurity

CS 410

3 Credits

Study of computer security, cryptography, and network security.

Topics Covered:

  • Cryptography and encryption
  • Network security protocols
  • Ethical hacking
  • Security policies and compliance
Prerequisite: CS 320 Offered: Spring

CS 420: Cloud Computing

CS 420

3 Credits

Study of cloud computing architectures, services, and deployment.

Topics Covered:

  • Cloud service models (IaaS, PaaS, SaaS)
  • Containerization and orchestration
  • Cloud security and compliance
  • DevOps practices
Prerequisite: CS 301, CS 320 Offered: Spring

Additional Information

Course Registration

Students must meet all prerequisites before registering for advanced courses. Registration is done through the student portal.

Academic Advising

All students are assigned an academic advisor to help with course selection and degree planning.

Course Rotation

Not all courses are offered every semester. Check the academic calendar for course availability.

Independent Study

Special topics courses and independent study options are available with faculty approval.
